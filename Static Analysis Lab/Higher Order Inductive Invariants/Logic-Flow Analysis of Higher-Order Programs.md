
Anything Hat is an abstract version of the concrete version

Abstract Numbers
- "0" -> "definitely zero"
- "1" -> "definitely one"
- inf -> "some number >= 2 (unknown, many possibilities)"

Ordering ⊑
⊑ = "is less precise than"

a⊑b means "b is at least as informative as a"

# Abstract Garbage Collection
Anything can't be referenced is collected.
This ensures precision by removing old data that pollutes the store
